Before You Generate
01 Ensure no row in Practitioner Name or Facility Name columns are both highlighted orange — this indicates a duplicate entry having same facility and practitioner name and needs to be deleted.
02 If a cell within the Facility Name column is highlighted orange, the facility may be intentionally covered by a backup practitioner. Verify before proceeding.
03 Confirm all Backup coverage types have an End Date set.
04 Review the Coverage Audit widget for facilities missing coverage and update the schedule as needed. If a correction has been made to the schedule, please refresh the page to update the coverage audit widget.
05 Ensure Weekly Coverage Schedule does not have blank cells in the Practitioner Name and Facility Name columns. A blank row at the end of the table is normal.
06 Please make sure rows that are highlighted red are addressed appropriately. A red highlight means the cell values in the Practitioner Name and Facility Name columns are not correctly linked to the master table. This could lead to discrepancy in numbers and improper reporting. If you come across one, double click on the Practitioner Name and/or Facility Name cell values and select the value from the dropdown.
07 If the values do not show up, please reach out to admin.
08 Make sure the week number and start of the week is correct. Please review the details here.
